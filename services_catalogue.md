### Service 1 — Data Quality Audit

**What it is:**
A structured assessment of your existing data across five dimensions —
completeness, uniqueness, validity, consistency, and business rule compliance.
We profile every column, identify every issue, and give you a clear picture
of the health of your data before any analysis begins.

**What is included:**
- Null and missing value analysis per column
- Duplicate record detection
- Data type and format validation
- Business rules validation — are values logically correct
- Statistical summary — min, max, mean, distribution
- Written findings report with severity ranking
- Recommendations for cleaning and improvement

**Who it is for:**
Any business that has data but has never formally checked its quality.
Common triggers — new data team being set up, preparing for a dashboard project,
regulatory audit preparation, or suspicion that existing reports are unreliable.

**Deliverable:**
Data quality report document and annotated notebook showing all findings.

**Indicative Pricing:**
Single dataset: Rs 25,000 — 40,000
Multiple datasets (3-5 tables): Rs 50,000 — 75,000

### Service 2 — Data Cleaning and Engineering

**What it is:**
Taking raw, messy data and transforming it into a clean, structured, analysis-ready
dataset. We handle currency formatting, date parsing, null treatment, deduplication,
derived column creation, and multi-table joins.

**What is included:**
- Full cleaning pipeline in Python and DuckDB
- Standardised formats across all columns
- Engineered derived columns — segments, tiers, flags, calculated fields
- Multi-table joins into a unified master dataset
- SQL scripts documenting all transformation logic
- Cleaned CSV exports
- Data dictionary explaining every column

**Who it is for:**
Businesses with data spread across multiple systems that has never been unified.
Common in fintech, healthcare, and operations where each function uses a
different tool and nobody has ever connected them.

**Deliverable:**
Clean master dataset, Python notebooks, SQL scripts, and data dictionary.

**Indicative Pricing:**
Single table: Rs 35,000 — 60,000
Multi-table (3-5 sources): Rs 75,000 — 1,50,000

### Service 5 — Anomaly Detection and Monitoring

**What it is:**
Building an automated monitoring system on your daily operational metrics.
We use rolling average methodology to establish normal baselines and flag
days where metrics deviate significantly — giving your team early warning
before problems become crises.

**What is included:**
- Metric selection and monitoring design
- Rolling average detection model — 7 day window, 2 standard deviation bands
- Spike and drop classification
- Severity scoring — deviation percentage
- Automated alert report generation
- Time series visualisations with control bands
- Documentation and handover

**Who it is for:**
Fintech companies, payment platforms, and operations teams who discover
problems reactively — hours or days after they occur. Ideal for teams
with daily transaction or ticket data.

**Deliverable:**
Monitoring notebook, daily alert report CSV, and dashboard charts.

**Indicative Pricing:**
Single metric monitoring: Rs 25,000 — 40,000
Full monitoring suite (3-5 metrics): Rs 60,000 — 1,00,000
